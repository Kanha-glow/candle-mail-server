# Candle Torque Atlas

Candle Torque Atlas is a brochure-style automotive catalog built with Next.js.

The project is focused on visual exploration instead of e-commerce:
- no cart
- no checkout
- no buy flow

## Catalog scope

The homepage includes famous flagship vehicles grouped by category.

Cars:
- Sedan Legends
- Hot Hatch Icons
- Performance SUVs
- Supercars and Hypercars

Bikes:
- Sport Bikes
- Naked Streetfighters
- Touring and Adventure

Brands covered include BMW, Audi, Porsche, Mercedes-AMG, Ford, Chevrolet, Pagani, Lamborghini, Ferrari, Yamaha, Ducati, Honda, and KTM.

## Run locally

```bash
npm install
npm run dev
```

Open `http://localhost:3001`.

## Tech stack

- Next.js App Router
- React + TypeScript
- Tailwind CSS
- Custom CSS animation and automotive theme tokens
